name: Parma Polyhedra Library
version: "1.0"
project-url: http://bugseng.com/products/ppl/

configuration-defaults:
 - INSTALL_PPL: true
 - PPL_PREFIX: /usr/local

PPL, the Parma Polyhedra Library is another mathematics library
that can be used to provide some optimizations in GCC.  It relies
on GMP.

If your system already has PPL installed, this step can be
skipped.

configure-commands:
 - ./configure --with-gmp=PARAM[GMP_PREFIX] --prefix=PARAM[PPL_PREFIX]
